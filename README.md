# Landing Page - Fotógrafo Trap/Rap

Landing page minimalista e moderna para fotógrafo/videomaker especializado em trap e rap.

## Stack

- React
- Vite
- TailwindCSS
- Lucide React (ícones)

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estrutura de Assets

Coloque as seguintes imagens na pasta `public/assets/`:

- `logo.jpeg` - Logo do fotógrafo
- `hero.jpeg` - Foto principal da seção Hero
- `FOTO1.jpeg` até `FOTO6.jpeg` - Fotos dos trabalhos
- `CLIPE1.jpeg` - Thumbnail do videoclipe
- `CTA1.jpeg` - Imagem da seção de contato
- `FOTOGRAFO1.jpeg` - Foto do fotógrafo para seção "Sobre Mim"
- `SILEO.png` - Logo da assinatura

## Características

- Design minimalista e elegante
- Paleta de cores dark urbana
- Animações suaves e discretas
- Totalmente responsivo (mobile-first)
- Navegação por âncoras
- Integração com Instagram e YouTube
